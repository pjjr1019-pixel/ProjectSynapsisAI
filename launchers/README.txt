Launchers
=========

This folder holds the user-facing launcher scripts for the portable/switchable app.

Use:
- Launch Mode Chooser.vbs
- Launch Mode Chooser.cmd

Optional tools:
- Launch Horizons Task Manager.cmd
- Switch to Dev Mode.cmd
- Switch to Portable Mode.cmd
- Show Launch Mode.cmd
- Set Dev Source Root.cmd

Older compatibility launchers are stored in:
- legacy\
