# Windows JS Skill Pack Overview

Lookup + execution pack for Windows settings/control panel/tools/folders.

- Canonical index: scripts/windows/INDEX.json
- Compact indexes: scripts/windows/registry/compact-skills.json and compact-aliases.json
- Runtime files: scripts/windows/tools, scripts/windows/executors, scripts/windows/skills
- Provenance: imports/windows-js-skill-pack/manifests

Coverage is broad but does not guarantee arbitrary deep UI automation.
