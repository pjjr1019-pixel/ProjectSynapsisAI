## Overview

Two complementary brain filtering packages have been integrated into the Horizons.AI repository:

1. **brain-filter-kit** — Manual CLI tool for deep filtering and analysis
2. **horizons-brain-startup-package** — Automatic background worker that runs on every app launch

Together, they provide:
- **On every app start:** Lightweight incremental brain sync (non-blocking)
- **On demand:** Deep brain analysis and filtering via CLI commands
- **Manual full rebuild:** Heavy rebuild when needed
- **Clean approved corpus:** Safe, curated knowledge for AI retrieval
